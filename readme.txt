An AJAXified Twitter feed widget.

== Description ==

Integrate an amazing **Twitter Feed Widget** to your widget's sidebar! Using Ajax, the data is being loaded "behind the scene" thus allowing the page to load quickly.

Feature List:

1. Use Ajax to load your tweets on the background without increasing load time
1. Graphically attractive, easily customizable, various styles are available
1. Very simple to setup
1. Specifically designed to work with both "ltr" and "rtl" languages (see screenshots)

== Installation ==

1. Download the plugin zip package and extract it.
2. Put the folder named "wd_twitter_feed" under /wp-content/plugins/ directory
3. Goto the plugins page in your Wordpress admin panel and click "Activate"
4. Goto your widget area (Appearance -> widgets) and add *WD Twitter Feed* to your widget panel

== Frequently Asked Questions ==

= What happens if I disable AJAX? =

The  widget will use php instead, slightly increasing load time (usually adds less than one second to load time)

= Where can i update the settings? / I can't find the plugin settings =

In order to update the settings of the widget, you'll need to login to your Wordpress admin panel and navigate to *Appearance -> Widgets* then choose *WD Twitter Feed* and drag it to the sidebar that you want to use it on.
