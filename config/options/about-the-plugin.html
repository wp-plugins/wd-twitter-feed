Askupa?? Twitter Feed plugin for Wordpress is a powerful tool for displaying twitter feeds in a wide array of ways. Using the plugin, one can:

  • Add a twitter feed widget - a fully featured twitter feed widget is available under the widgets section
  • Add twitter feeds with shortcodes - for your convinience, a shortcode editor was added to the post editor. By clicking the blue bird button, you can easily add tweets to your posts
  • Display tweets as a simple list, scrolling list and sliding list
  • Style the tweets using various pre-made skins
